# 🤖 Roboschool AI Chatbot

Roboschool o'quv markazi uchun sun'iy intellekt yordamchisi.

## 🚀 Xususiyatlari

- ✨ ChatGPT uslubida streaming text
- 🎭 Jonli AI effektlari (o'ylash, pauzalar)
- 💬 Emoji reaktsiyalar
- 📝 Ro'yxatdan o'tish tizimi
- 📱 Telegram/Instagram integratsiyasi
- 🌙 Chiroyli dark tema

## 📚 Kurslar

- 🧱 Lego WeDo (6-8 yosh)
- 🤖 Robototexnika (8-15 yosh)
- 💻 Dasturlash (10-18 yosh)
- 🧠 Sun'iy Intellekt (16+ yosh)
- 🎨 3D Modellashtirish (10-18 yosh)

## 🛠 O'rnatish

```bash
# Kutubxonalarni o'rnatish
npm install

# Development serverini ishga tushirish
npm run dev

# Production uchun build qilish
npm run build
```
